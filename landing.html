100% Offline Capable

Operates over a local Wi-Fi hotspot network — no internet or cellular data required during active game play. Full scoreboard, overlays, and shot clock all work offline.

Dual-Cloud Redundancy

Two independent cloud deployments — Vercel + Firebase (serverless) and OnRender (full-stack). If one goes down, switch to the other without missing a beat.

Zero Clock Drift

Clocks tick authoritatively — on the server in Node.js mode, and in the operator's browser in Vercel/Firebase mode — preventing drift across all connected displays.

Firebase Realtime Sync

In Vercel mode, all score changes, clock ticks, and buzzer events are synced instantly through Google Firebase — no backend server required.

Auto Brand Colors

Uploading any team logo automatically extracts its dominant color, updating scoreboard styling across all screens instantly.
